How to use the BOX

BOX is kind of communication and file sharing application with multiple users. It sends every message and data to all the users connected with the same server. It is like a messenger, whatsapp, we-chat, IMO, share-it or other communication and sharing application. You can do group chat, you can share your files, you can do anything you like to do. It works though INTERNET safely. It communicates though dark web. You can use masked IP with proxy or VPN. You can ensure your security and privacy with it.


To use this application you have to connect with a Server first. You can make your own server by running "BOX Server" application. Or you can connect with others server IP and Port. You will also need a password to verify your identity to other users connected with the server.


When you host a server you will see the following Tray Icon[shown at Fig-1 inside a green Box]. When you put your mouse there it will show the status of the running server. And it will pop up a dialog too. You can customize it too.[in the red area]
